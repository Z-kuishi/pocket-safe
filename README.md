# 🔐 口袋保险箱

让加密像拖拽一样简单！

---

## ⚠️ 重要提醒

**请务必牢记您的密码！**

- ❌ 本软件不会存储您的密码
- ❌ 本软件不会提供密码找回功能
- ❌ 忘记密码 = 文件永久无法解密

## ✨ 功能

- 🖱️ **拖拽加密**：拖入文件/文件夹 → 自动加密
- 🖱️ **选择加密**：选择文件/文件夹 → 自动加密
- 🔓 **一键解密**：选中 `.pse` 文件 → 输入密码 → 还原
- 🔒 **AES-256-GCM**：256位密钥
- 🏠 **纯本地运行**：不上传任何数据，隐私安全
- 📂 **自定义文件夹**：支持任意目录加密/解密
---
🔐 加密逻辑
本软件采用 **AES-256-GCM** 加密，结合 **PBKDF2** 密钥派生算法。
